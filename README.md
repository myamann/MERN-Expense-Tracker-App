# MERN-Expense-Tracker-App

improved my expense tracker app. I added some new features like dark mode.
	
## Technologies
Project is created with:
* ReactJs
* Nodejs
* Express
* MongodB
	
## Setup
To run this project, install it locally using npm:

```
# Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
